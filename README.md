# Padrino::Jbuilder

Adds the jbuilder templating engine to Padrino

## Installation

Add this line to your application's Gemfile:

    gem 'padrino-jbuilder'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install padrino-jbuilder
